5 markdown to json -- 用Markdown方式定义数据，然后转换成json https://github.com/njvack/markdown-to-json

有时候，你需要编写json
手工编写，非常痛苦，因为它有一个繁琐的格式。
构建一个工具来编写，非常麻烦，因为你要提前准备一堆的工作。

其实，你想要的并不复杂。你只需要打开文本编辑器，以适合普通人阅读的方式，处理嵌套的数据。
markdown-json 便是为此而生

11 Typography.js -- 强大的工具包，用于构建漂亮的网站
https://github.com/KyleAMathews/typography.js

网站的排版设计是一个复杂的系统，因为多个元素相互关联。
试想，数十个元素的一百多个样式，必须和谐有序。一次的修改意味着数十次繁琐的重新计算与调整。
Typography提供了一种探索和定义排版设计的方法。
事先提供基本配置，Typography引擎为相关元素生成css。
